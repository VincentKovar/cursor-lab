# Fog Crossing

A survival exploration game where fog conceals the way forward. Navigate carefully, and find your path through.

## Play Now

**[Play Fog Crossing](https://vincentkovar.github.io/cursor-lab/Fog%20Crossing/)**

Works best on mobile. Add it to your home screen for the full app-like experience:

- **iPhone (Safari):** Tap the Share icon, then "Add to Home Screen"
- **Android (Chrome):** Tap the menu (⋮), then "Add to Home Screen" or "Install App"

## Controls

- **Tap** to move or interact
- **Swipe** to navigate through the fog

## About

Built by Vincent Kovar using Claude Code (Fable 5). Fog Crossing is part of an ongoing collection of experimental games and prototypes.
